# zrx\_repairkit

## An advanced repairkit system for FiveM

### Features

* Item based
* Allowed Classes/Vehicles/Jobs
* Highly configurable
* Synced between all players
* Protected Events
* Discord log with many information
* Update checker
* Optimized
* Synced
* 0.0 ms on idle
* 0.0 ms while in use

### Requirements

* zrx\_utility (latest)
* ox\_lib (latest)
