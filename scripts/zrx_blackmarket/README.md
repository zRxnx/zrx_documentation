# zrx\_blackmarket

## An advanced blackmarket system for FiveM

{% embed url="https://github.com/zRxnx/zrx_blackmarket" %}

### Features

* Random locations
* Alert system
* Blocked Jobs
* Highly configurable
* Protected Events
* Discord log with many information
* Update checker
* 0.0 ms on idle
* 0.0 ms while in use

### Requirements

* zrx\_utility (latest)
* ox\_lib (latest)
* ox\_target (latest)
