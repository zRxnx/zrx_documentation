# zrx\_uniqueid

## An advanced unique identifier system for FiveM

### Features

* Menu to edit exisiting UIDs
* Discord log with many information
* Update checker
* Optimized
* Synced
* 0.0 ms on idle
* 0.0 ms while in use

### Requirements

* zrx\_utility (latest)
* ox\_lib (latest)
* oxmysql (latest)
