# zrx\_panicbutton

## An advanced panicbutton system for FiveM

{% embed url="https://github.com/zRxnx/zrx_panicbutton" %}

### Features

* Template system
* Multiple panicbutton support
* Notifications
* Discord log with many information
* Update checker
* Optimized
* Synced
* 0.0 ms on idle
* 0.0 ms while in use

### Requirements

* zrx\_utility (latest)
* ox\_lib (latest)
