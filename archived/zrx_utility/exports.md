# Exports

## Client

### getUtility()

Returns the core Utility

<mark style="color:green;">**Returns:**</mark>\
`table` - Utility object

<mark style="color:orange;">**Example:**</mark>

```lua
local CORE = exports.zrx_utility:getUtility()

print(json.encode(CORE, { indent = true })
```
